html 5 stuff

by adnan

basic changes of HTML 5 (from 4.01):
doctype is now:
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
<!doctype html>
content meta tag:
<meta http-equiv="content-type" content="text/html; charset=UTF-8">
<meta charset="utf-8">
link tag for css:
<link type="text/css" rel="stylesheet" href="css/blah.css">
<link rel="stylesheet" href="css/blah.css">
script tag for js:
<script type="text/javascript" src="js/blah.js"></script>
<script src="js/blah.js"></script>
or
<script>
     alert("great scott!");
</script>
